# Titanic Dataset - Data Cleaning & Exploratory Data Analysis (EDA)

## Project Overview

This project performs Data Cleaning and Exploratory Data Analysis (EDA) on the Titanic Dataset using Python.
The goal of the project is to analyze passenger information and identify patterns and factors that influenced survival during the Titanic disaster.

## Dataset Used
Titanic Dataset

The dataset contains:
- Passenger demographics
- Ticket details
- Passenger class
- Fare information
- Survival status

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Data Cleaning Steps

✔ Handled missing values  
✔ Removed duplicate rows  
✔ Dropped unnecessary columns  
✔ Checked data types  
✔ Performed feature engineering

## Exploratory Data Analysis

The following visualizations were created:

- Survival Count Plot
- Gender vs Survival
- Passenger Class vs Survival
- Age Distribution
- Fare Distribution
- Correlation Heatmap
- Fare vs Survival
- Family Size vs Survival
- Pairplot Analysis
- Survival Percentage Pie Chart

## Key Insights

- Female passengers had higher survival rates.
- First-class passengers survived more than third-class passengers.
- Higher ticket fares increased survival probability.
- Small families had better survival chances.
- Passenger class and fare strongly influenced survival.
